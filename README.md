#PowerBI-Dashboard
  Sales Analysis Dashboard built in Power BI to deliver insights on revenue, profit, regional performance, and product trends. Features interactive visuals, KPIs, and filters for dynamic analysis, enabling data-driven decision-making and improved business performance.
Sales Analysis Dashboard
Overview:
  An interactive Sales Analysis Dashboard built using Power BI to track revenue, profit, and sales performance across regions and products.

Features:
  KPI tracking (Sales, Profit)
  Regional and product analysis
  Time-based trends
  Interactive filters and visuals
  
Tools:
  Power BI, Power Query, DAX

Usage:
  Download the .pbix file and open it in Power BI Desktop to explore the dashboard.
